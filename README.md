# 🔍 Tech0 Search v0.1

PROJECT ZERO ─ 社内ナレッジ検索エンジン（Week2版）

## 概要

社内ナレッジを検索できるWebアプリのプロトタイプです。  
Streamlit を使用して検索UIを実装しています。

## 実装ステータス

- Week1：API取得・スクレイピング
- Week2：検索UI + 手動登録 + Git管理

## 機能

- 🔍 キーワード検索（部分一致）
- 📝 ページの手動登録
- 📋 登録ページ一覧表示

## 使用技術

- Python 3.10
- Streamlit
- JSONデータ管理
- Git / GitHub

## セットアップ

```bash
git clone https://github.com/MogiSatoru-7/tech0-search.git
cd tech0-search
pip install -r requirements.txt
streamlit run app.py
```
